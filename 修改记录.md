## 2026-06-23 · 第九轮 · 放大图缩略图列bug + 字幕叠放 + 档案卡片改版（v69→v70）

- **修缩略图列偏移 bug**：放大后切换不同宽度的照片时，图片宽度有 260ms 过渡动画，缩略图列原来按动画中途的位置定位，导致列偏移、不贴在图右边。改成按「图片最终宽度」(viewport中心+半宽) 计算，并给列加同样 260ms 过渡，跟着图一起滑到位。
- **详情页补充/细节图字幕叠放**：原来字幕在图下方，现改为叠放在图片右下角，和放大图字幕同款（白字+阴影）。
- **字幕字号 +2pt**：叠放字幕统一用 `calc(11px + 2pt)`。
- **档案卡片改版**：① 圆角边框只包住图片，ID 移到图片下方、左对齐、无边框背景；② 多图/插图标志移到图片右下角；编辑按钮挪到右上角避让。

## 2026-06-23 · 第八轮 · 详情页/放大图/左栏微调（v68→v69）

- **详情页主图对齐**：竖图原来被装进更宽的框里居中（左右各空约 53px），导致 ID 和正文比照片宽。改成封面图始终铺满栏宽（去掉高度上限的留白），文字自然与照片左右对齐。
- **放大图缩略图列**：从屏幕最右边改到「放大图右侧 15px」处（JS 按图片实际右边缘定位，自动随图宽变化）。
- **放大图切换更顺**：进入放大时预加载所有可放大图；点缩略图切换时只挪高亮、不再整列重建，去掉卡顿/闪烁。
- **空白处滑动切换**：放大状态下在图外空白处左右拖动可上一张/下一张（左滑下一张，右滑上一张，循环）；普通点击空白仍是关闭。
- **放大图显示文字**：放大图右下角加「标题, ID, 时间」小字幕（白字+阴影，照片上看得清），样式同详情页每张图的右下角小字；封面放大也显示。
- **左栏标题同行**：修 `.location-button span{display:block}` 优先级压过 idrow 的 flex 导致标题换行；标题现与 ID 同一行。
